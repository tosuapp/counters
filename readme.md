# The official tosu repository of overlays

A curated collection of community-made overlays available in the tosu dashboard.

# How to use

> [!WARNING]
> This method is discouraged and should only be used if you can not download them from the tosu dashboard.

### Follow these steps:
1. Clone this repository.
2. Move the folders of the desired overlays into your tosu's `./static` directory.

You can also individually download each overlay by using the download buttons in the [counters' readme](https://github.com/tosuapp/counters/blob/master/counters/readme.md).
