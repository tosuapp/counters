# The official tosu! pp counters repository

A curated collection of community-made pp counters available in the tosu! dashboard.

# How to use
1. Clone this repository.
2. Move the desired pp counter folders into your tosu! `./static` directory.

> [!WARNING]
> This method is discouraged and should only be used if you can not download them from the tosu! dashboard.
